Projeto JavaScript


Seja bem-vindo ao Projeto JavaScript, um projeto desenvolvido por alunos do Ifal Campus Palmeira dos Índios - AL. O projeto foi criado, com a finalidade de práticar os conhecimentos passados em sala e como atividade avaliativa. Fomos desafiados a resolver 7 desafios de js, que você pode encontrar o resultado logo abaixo.

Questões

Questão 1

Cálculos matemáticos: média, moda e mediana

Questão 2

Cálculos matemáticos

Questão 3

Qual número é maior/menor e qual signo ele representa.

Questão 4

Verificador de idades

Questão 5

Verificação de altura, peso, data de nascimento e sexo

Questão 6

Embaralhando conjunto de nomes

Questão 7

Verificador de datas.
